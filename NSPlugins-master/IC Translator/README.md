### Plugin Information ###
**Name**: IC Translator

**Author**: Black Tea(rebel1324) and Google Translator

**Description**: This plugin allows you to say different language in IC. [>> Example Image](http://i.imgur.com/2OlrcO0.png)
You can easliy add additional languages with modifying the plugin. 
!! Default Original Language is English.
[>> Country(Language) Codes](https://sites.google.com/site/tomihasa/google-language-codes)
Default Languages: Korean(/kor), Japanese(/jap), German(/ger), Vietnamese(/vet), Russian(/rus), French(/fre)

### Installation ###
To install this plugin, go to gamemode/Your Schema Name here/plugins/ and extract download.zip in to the folder there. 
Does not requires new folder.
